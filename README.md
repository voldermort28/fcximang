## Install From Source

First, clone the project:

```bash
$ git clone <link><my-project-name>
```

```bash
$ cd <my-project-name>
$ npm install
```

## Running Your Local Server With Gulp

```bash
$ npm run dev
```

## Build Files

```bash
$ npm run build
```
